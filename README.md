## How To Compile And Run

Use VS Code task:
- compile
- compile and run

Or use terminal commands:

    g++ -std=c++17 -o program path/to/file.cpp
    ./program

With input and output files:

    g++ -std=c++17 -o program path/to/file.cpp && ./program < input.txt > output.txt

## Problem-Solving Template

For each problem, try this order:

1. Understand constraints and edge cases
2. Write brute force idea
3. Optimize approach
4. Dry run with sample
5. Code and test
6. Note key learning in comments
